All information on this page is extracted from sources that are public available on the internet.
Most firmware files cannot be distributed here due to their license. See also a post on the OpenWrt mailing list regarding this topic.

In November 2015 Lantiq started providing the first xDSL firmware binaries with a license that permits redistribution.

If you have seen a new firmware and want to get this table updated then please create a ticket with information how to extract the firmware file from a public source.
